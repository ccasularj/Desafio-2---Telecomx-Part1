# TelecomX - Churn Analysis

## Objective

Analyze customer churn in a telecommunications company.

## Dataset

Customer dataset including:

- demographics
- services
- billing
- churn status

## Key Insights

- Month-to-month contracts show higher churn
- Customers with shorter tenure cancel more
- Higher monthly charges increase churn risk

## Tools

Python  
Pandas  
Matplotlib  
Seaborn
